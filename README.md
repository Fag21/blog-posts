# Developer Blog Repository

This repository contains blog posts written in Markdown format. The posts are used as a content source for my personal portfolio website.

Each blog is stored as a Markdown (`.md`) file inside the `posts` directory. The portfolio fetches these files and displays them dynamically.

## Repository Structure

blog-repo
├── posts
│ ├── web-vs-mobile.md
│ ├── another-post.md
│
└── README.md

## Purpose

The goal of this repository is to:

- Store blog posts in a simple Markdown format
- Keep blog content separate from the portfolio code
- Make it easy to update content by pushing new posts to GitHub
- Allow the portfolio website to fetch and display posts dynamically
